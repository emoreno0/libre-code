# 💻 Libre Code 
### 🚧 Work in Progress

## 🧩 An open-source code editor
No telemetry. No subscriptions. No unnecessary features.  
Just a fast, editor built for real developers.

---

## ⚙️ Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/emoreno0/libre-code.git
cd libre-code

# 2. Install dependencies
npm install

# 3. Run it
npm run dev
```
